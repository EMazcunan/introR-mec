3.3   Variables

Igual que en Maxima utilizábamos el símbolo : para crear y asignar valores a variable, el operador de asignación en R es <- (compuesto por el símbolo menor < y un guión central -). La instrucción

x <- 2

crea una variable u objeto de nombre x y le asigna el valor 2 .

Una vez definida la variable x podemos referenciarla en cálculos posteriores. Por ejemplo, al ejecutar la instrucción

x^2-3

obtendremos la salida 1 (resultado de x 2 − 3 = 2 2 − 3 = 1 ).

Cuando se ejecuta una instrucción en la que se realiza un cálculo y se almacena en un objeto, la salida no muestra el valor almacenado:

y <- x^2-3 # no muestra el valor de y

La instrucción anterior crea el objeto y y le asigna el valor del cálculo x^2-3, pero no se muestra dicho valor en la salida. Para ver el valor almacenado en una variable hay que consultarlo explícitamente, utilizando la función print o simplemente escribiendo su nombre:

print(y) # muestra el valor de y
y
