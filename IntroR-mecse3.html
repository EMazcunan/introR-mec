3   Primeros pasos con R

Ahora que ya estás familiarizado con RStudio y sabes utilizarlo para escribir y ejecutar código de R, puedes empezar a aprender el lenguaje R. En esta sección se introducen las primeras nociones.

Para empezar, crea un nuevo script desde RStudio (File > New File > R Script) y guárdalo en tu equipo, en la ubicación que quieras, y con el nombre que quieras. Utiliza el script creado para escribir y ejecutar el código que encuentres conforme leas las explicaciones de los siguientes apartados (resaltado con fondo gris). Añade los comentarios que creas oportunos, y experimenta, escribiendo el código que te apetezca para probar las ideas que te vayan surgiendo.
