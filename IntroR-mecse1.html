1   ¿Qué son R y RStudio?

R es el lenguaje líder para análisis estadísticos y gráficos.

Se distribuye gratuitamente bajo la licencia GNU-GPL, y funciona en cualquier sistema operativo.

El código de R para llevar a cabo un análisis estadístico se guarda en un archivo de texto o script. Esto permite retomar o modificar el trabajo en una fecha posterior, y compartirlo fácilmente con otros.

Las funcionalidades de R pueden ampliarse creando paquetes, que definen nuevas funciones para propósitos específicos. Actualmente, R tiene más de 10000 paquetes publicados en el repositorio CRAN (Comprehensive R Archive Network), escritos por sus usuarios, que cualquiera puede instalar y usar libremente.

Como resultado de todo lo anterior, R tiene una gran comunidad de usuarios y desarrolladores, activa y creciente. Si necesitas una función para un propósito específico, es muy probable que alguien la haya creado antes. Y si tienes alguna duda sobre la sintaxis de R, casi seguro que encuentras la respuesta online fácilmente (etiqueta r en stackoverflow).

Como muestra de la popularidad de R, citamos algunos ejemplos de proyectos que lo usan:

  • La Agencia Nacional de Metereología ha desarrollado un paquete de R llamado climatol, para el tratamiento de series climáticas (más información aquí, para localizar la información en la página anterior busca el texto paquete “Climatol”).
  • Un sistema para detectar casos de intoxicaciones alimentarias en la ciudad de Chicago, mediante el análisis automatizado de tweets (más información aquí).

Para más información sobre R, visitar la página web del proyecto.

RStudio es una IDE (Integrated Development Environment) para R.

Está disponible gratuitamente, para los sistemas operativos más usuales.

Incluye una consola, un editor con resaltado de sintaxis y que permite ejecución directa del código, e integra herramientas para la visualización de conjuntos de datos y gráficos, acceso a la ayuda de R, etc.

Para más información sobre RStudio, puedes visitar la página web del proyecto.
